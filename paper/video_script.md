# Two-minute video — placeholder script

> **Status.** A script *shape*, not a script. It has no footage yet because no analysis has been performed yet. Drafting begins when Stage 4 of `EXECUTION_PLAN.md` is underway and the headline figure exists.

A two-minute video is two pages of double-spaced narration at most. Less is better.

## Four-beat structure (working assumption)

**Beat 1 — The question (15 s).** One sentence stating what is being asked. One sentence stating why the answer is hard. Camera: still title card over an orca call spectrogram.

**Beat 2 — The data (20 s).** A sentence on the dataset. A sentence on the encoder. One quick visual of the public-data provenance, with the dataset citation on-screen.

**Beat 3 — The result (60 s).** The headline figure on screen, animated *once*. The result stated in one sentence. The counterfactual control shown for ten seconds. The result re-stated, this time with the confidence interval.

**Beat 4 — Limitations (25 s).** Three short sentences: what the result does *not* imply. Whose work it builds on. What would be needed before any real interspecies communication claim could be made.

## What this script will *not* contain

- The phrase "Doctor Dolittle".
- The phrase "we talked to whales".
- Any mood music dramatic enough to overwhelm the orca audio.
- Any visualisation that has not been generated by code that lives in this repository at submission time.
