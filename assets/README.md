# Assets

This folder holds visual assets referenced by `README.md` and the Gradio Space.

* `banner.png` — README banner. Generate or replace with the preferred header image.
* `demo/` — small audio samples used by the Gradio demo's example buttons.

Image generation guidelines: dark editorial palette (`#0d1117` background, `#c9a84c` accent, `#e8e0d4` text, Lora display font); subject is an orca silhouette with an embedded spectrogram or call-type sigil — *not* a generic stock photo.
