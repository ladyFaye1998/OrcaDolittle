# Literature Review

This file is a compact map of the sources used by the repository. Full BibTeX entries live in `paper/refs.bib`.

## Data And Provenance

- DCLDE 2026 provides the public killer-whale acoustic substrate and metadata [@palmer2025dclde; @palmer2025dclde_data].

## Encoders

- AVES2 provides the current implemented frozen-embedding path [@hagiwara2023aves; @chen2022beats].
- NatureLM-audio is the documented comparative encoder path [@robinson2024naturelm].

## Repertoire And Clustering

- Sainburg et al. provide the core unsupervised repertoire-discovery precedent [@sainburg2020].
- UMAP and HDBSCAN provide the dimensionality-reduction and density-clustering tools [@mcinnes2018umap; @mcinnes2017hdbscan].

## Orca Acoustic Context

- Ford, Foote, Filatova, Riesch, and Yurk provide call-type, dialect, and behavioural-context grounding [@ford1989; @foote2008; @filatova2015; @riesch2008; @yurk2002].

## Sequence Modelling

- Transformer and masked-language-model foundations are from Vaswani et al. and Devlin et al. [@vaswani2017attention; @devlin2019bert].
- Cetacean sequence-modelling context is informed by sperm-whale coda work [@sharma2024].
