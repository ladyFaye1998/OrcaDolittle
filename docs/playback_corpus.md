# Playback Corpus Notes

This file tracks peer-reviewed playback literature that may support future response-association analyses. The active H4 head is provider/site control; playback extraction is kept as a literature task until per-trial response data are available from source papers or supplementary material.

## Inclusion Requirements

1. Peer-reviewed source.
2. Acoustic stimulus and listener species are clearly described.
3. Response statistics are reported per trial, per condition, or in a figure that can be digitized responsibly.
4. The stimulus type can be linked to a documented call category or acoustic class.
5. Extraction uncertainty can be footnoted in the manuscript.

## Current Boundary

Do not report a learned playback-response result from this file alone. Until per-trial values are extracted and audited, these papers support literature context and future extraction planning only.

## Extraction Checklist

- Confirm full citation and DOI.
- Record stimulus class, listener species, sample size, response metric, and response direction.
- Separate within-species and cross-species evidence.
- Store extracted values in a small table with one row per trial or condition.
- Cite source keys from `paper/refs.bib` in every row.
