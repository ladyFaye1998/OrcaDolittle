One dictionary for call units, examples, and evidence.

This is not a translation dictionary. It is the project map: real FEROP catalogue samples, larger DCLDE model-label inventory, synthetic listening sketches, and the field-test language needed to ask whether any unit has receiver-interpretable content.

Current word count: 0 validated orca words. The dictionary lists acoustic units and hypotheses. Human shorthand like "food is here" is an analogy for field-test design, not a claim that the call has been translated.

What the dictionary can say

It can say this is a named call type, this sample is a real catalogue recording, this synthetic file is a contour sketch, and this evidence supports a field-test design.

safe claim

What it cannot say yet

It cannot say K1 means hello, K7 means come here, or any other one-to-one word mapping. That is what a content-controlled playback study must test.

Glossary

WordA receiver-interpretable unit with content. Current count in this project: zero validated words.
Call typeA stereotyped acoustic category from a catalogue or model head. FEROP K-types and DCLDE labels are call types.
VariantA catalogue example or subform within a call type, such as K5i, K5ii, or K7X.
DialectGroup- or pod-linked call structure. The strongest playback response evidence here is dialect-sensitive response.
Production contextWhat the vocalizing whale was doing when the call was recorded. This is not the same as what a listener understands.
Receiver responseWhat a whale does after hearing a broadcast. This is the evidence needed to move from call structure toward meaning.
Synthetic contourA generated audio sketch for listening and UI demonstration. It is not a validated orca vocalization.
Field kitThe protocol, run sheet, scoring sheet, and stimulus manifest used to test the next receiver-response question.
